## üõ†Ô∏è Common Tasks

### Change Colors
Edit `tailwind.config.ts`:
```typescript
colors: {
  primary: {
    DEFAULT: "#YOUR_COLOR",
  },
}
```

### Add a Page
1. Create `app/new-page/page.tsx`
2. Add content
3. Link in header

### Update Navigation
Edit `components/header.tsx` to add/remove links.

### Add Images
Place in `public/` or use external URLs.

### Deploy to Vercel
```bash
vercel login
vercel --prod
```
